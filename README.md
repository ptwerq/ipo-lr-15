# Лабораторная работа №15 — Простейший веб-сервис на Flask

## Задание

Создать простейший веб-сервис на Flask с маршрутами:

1. `/` — выводит приветствие  
2. `/hello/<name>` — персональное приветствие  
3. `/square/<int:number>` — возвращает квадрат числа  

Индивидуальное задание (Вариант 1):

1. `GET /sum` — принимает `a` и `b` через query-параметры, возвращает сумму  
2. `GET /info` — возвращает JSON с полями: `version`, `author`, `start_date`

---

## Ссылка на сервер

[http://127.0.0.1:5000](http://127.0.0.1:5000)

---

## Примеры маршрутов и ответов

- Главная страница: [`/`](http://127.0.0.1:5000/)  
**Ответ:** Привет! Это простейший веб-сервис на Flask.
  - Персональное приветствие: [`/hello/Taya`](http://127.0.0.1:5000/hello/Taya)  
**Ответ:**  Привет, Taya!
- Квадрат числа: [`/square/5`](http://127.0.0.1:5000/square/5)  
**Ответ:**  Квадрат числа 5 равен 25
  - Сумма чисел: [`/sum?a=3&b=7`](http://127.0.0.1:5000/sum?a=3&b=7)  
**Ответ:**  Сумма 3 + 7 = 10
- Информация JSON: [`/info`](http://127.0.0.1:5000/info)  
**Ответ:**  
```json
{
    "version": "1.0",
    "author": "Тая Полковникова",
    "start_date": "2025-12-18 20:00:00"
}
